# Student Management System

A Python-based Student Management System built using Object-Oriented Programming (OOP) concepts and JSON file handling.

## Features

- Add Student
- Search Student
- Update Student Marks
- Delete Student
- Store Records in JSON
- Unique Roll Number Validation

## Technologies Used

- Python
- OOP
- JSON
- File Handling

## Project Structure

```
Student_System_management/
│
├── students/
│   ├── file_handling.py
│   └── ...
│
├── README.md
└── main.py
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Shaik-Mahammad-Arif/Student_System.git
```

2. Navigate to the project folder

```bash
cd Student_System
```

3. Run the application

```bash
python main.py
```
